# API

The API area is organized around package families and user goals, not namespaces.

The API pages assume you know which layer you need: core substrate, Avalonia
integration, Projection, or ElmView.

Package names follow that split:

- `Elmish.Glue.Core` contains framework-neutral substrate.
- `Elmish.Avalonia.Glue` contains Avalonia runtime integration.
- `Elmish.Avalonia.Glue.Projection` contains explicit viewmodel helpers.
- `Elmish.Avalonia.Glue.ElmView` contains F# view-record host helpers.
