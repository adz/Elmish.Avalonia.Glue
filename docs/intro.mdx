# Elmish.Avalonia.Glue

Avalonia-first Elmish integration with two deliberate authoring families:

- `Projection` for explicit CLR-facing viewmodels
- `ElmView` for immutable F# view records and a thin generated bindable surface

The site is organized around user intent, not namespace taxonomy. Start with the guides, then use the API hubs when you need the member map for a package family.
